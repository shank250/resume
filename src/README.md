This directory is aimed to pick up template_resume_job_role and then based on some Job descriptions query from the user or parameter it would create on the fly new latex code for the specific job description. However, we would be keeping all the job applications newly created resume under a new folder but file name with username_temp_name_abc.pdf where temp_name would come from the selected template and abc would be some unique hash which would be used to map multiple resume applications within same company.
